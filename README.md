Math Module.
============

The Euclidean Distance:
------------------------
`````ruby
from math import *
x1 = float (input())
y1 = float (input())
x2 = float (input())
y2 = float (input())

p = sqrt ((pow(x1-x2,2)) + (pow(y1-y2,2)))

print(p)
`````
Circle:
------
```````````ruby
from math import *
R = float(input())
p = 3.141592653589793
S = p*pow(R,2) 
C = 2*(p*R)

print(S)
print(C)
````````````````
